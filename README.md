# SentinelShield-WAF
A Python-based Intrusion Detection and Web Protection System simulating a lightweight Web Application Firewall (WAF).
🛡 SentinelShield – Intrusion Detection & Web Protection System

SentinelShield is a simplified Web Application Firewall (WAF) and Intrusion Detection System built using Python and Flask.

The system inspects HTTP requests, detects malicious payloads, logs suspicious activity, and blocks common web attacks.

🔍 Features

SQL Injection Detection

Cross-Site Scripting (XSS) Detection

Directory Traversal Detection

Rule-Based Signature Engine

Attack Logging System

HTTP Request Inspection

🧠 Security Concepts Implemented

Signature-Based Detection

HTTP Request Analysis

Web Application Firewall (WAF) Simulation

Intrusion Detection Workflow

Logging & Alerting

🛠 Technologies Used

Python

Flask

Regular Expressions (re)

pip install flask
python app.py
